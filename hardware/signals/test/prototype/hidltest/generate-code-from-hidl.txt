Generate code from HIDL interface
---------------------------------
1. Source your android anve≈ïironment:
    . build/envsetup.sh
    lunch
2. Goto your root:
    croot
3. Build hidl-gen if not already done:
    make hidl-gen
4. Generate C++ code from hidl files, example:
    hidl-gen -o hidl-generated/ -L c++ -r vcc:interface/vcc -r android.hidl:${ANDROID_BUILD_TOP}/system/libhidl/transport vcc.vcctest@1.0

Note! It is important the defined package name defined in interface description(*.hal files) match the folder structure in which it is placed.

Building and running
--------------------
1. Start the docker image:
    vendor/volvocars/tools/docker_build/run.sh
2. Build the server:
    cd vendor/volvocars/hardware/signals/test/prototype/hidltest/server
    mm
3. Build the client:
    cd vendor/volvocars/hardware/signals/test/prototype/hidltest/client
    mm
4. Download to target:
    adb remount
    adb push out/target/product/ihu_vcc/system/bin/hidl-test-service /system/bin
    adb push out/target/product/ihu_vcc/system/bin/hidl-test-client /system/bin
5. Running server:
    adb shell
    cd /system/bin
    ./hidl-test-service
6. Running client:
    adb shell
    cd /system/bin
    ./hidl-test-client

If you run "adb logcat hidl-test-server@1.0-service:I hidl-test-client@1.0:I *:S" from host you should see output from hidl test server and client in output.
